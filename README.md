# HopeDraw

HopeDraw is a lightweight, browser-based vector drawing application designed for creating diagrams, shapes, and mathematical illustrations. It operates entirely on the client side, leveraging HTML5 Canvas and SVG technologies to provide a fast and responsive drawing experience.

## Features

- **Rich Shape Library**: Draw rectangles, ellipses, arcs (with pie, chord, and partial stroke support), lines (with arrowheads, curves, and elbows), and parallelograms.
- **Path Editing**: Create custom shapes using the Pen tool, and fine-tune Bézier control points and anchors with the dedicated Path node editor.
- **Text & Typography**: Add text nodes with comprehensive styling options, including a variety of web-safe fonts, alignments, and colors.
- **Mathematical Equations**: Seamlessly embed LaTeX formulas directly into your diagrams natively rendered via KaTeX.
- **Layer & Z-Index Management**: Arrange components front-to-back and organize your canvas using a dedicated Layers panel.
- **Auto-saving**: HopeDraw automatically saves your work constantly to your browser's Local Storage, meaning you never lose your progress during rapid interactions.
- **Copy & Paste**: Duplicate shapes and layers quickly using standard keyboard shortcuts (Ctrl+C / Ctrl+V) with full undo/redo historical support.

## User's Guide

### Getting Started
1. Run a local web server in the project directory (e.g., `python3 -m http.server 9000`).
2. Open `http://localhost:9000` in your web browser.
3. Use the left toolbar to select drawing tools. The properties of the currently selected tool or shape will appear in the right-hand Properties Panel.

### Interacting with Shapes
- **Selection**: Use the **Select Tool** (arrow icon) to click on shapes. Once selected, an 8-point bounding box will appear around the shape, allowing you to scale and rotate it.
- **Advanced Editing**: Double-clicking on certain shapes unlocks advanced editing modes:
  - **Lines**: Double-clicking a line allows you to independently drag its start and end anchor points.
  - **Paths**: Double-clicking a custom SVG path opens the node editor, allowing you to manipulate specific Bézier control points and segment joints.
- **Properties**: With a shape selected, use the right sidebar to change its fill color, stroke weight, opacity, and shape-specific settings (like the sweep angle of an arc, or the arrowheads of a line).
- **Formulas**: To add a math equation, select the Formula tool (the `$$ \Sigma $$` icon), click on the canvas, and use the Properties panel to edit the underlying LaTeX source code.

### Managing Projects
- Your work is automatically saved to your browser's local storage after every interaction.
- Click the **Project Menu** (hamburger icon) in the top left to Create, Open, or Manage multiple local projects.

## Limitations and Known Issues

While HopeDraw provides a robust editing environment, it has a few architectural limitations to be aware of:

1. **Equation Rendering in Exported SVGs**: 
   Mathematical equations are rendered using the HTML5 `<foreignObject>` tag combined with the external KaTeX library. While this looks perfect inside a web browser, standard standalone SVG viewers and desktop vector editing software (like Adobe Illustrator or Inkscape) rarely support HTML/CSS execution inside `<foreignObject>`. As a result, imported SVG files containing formulas will likely appear blank or broken when viewed outside of a web browser.
2. **SVG Import Compatibility**: 
   HopeDraw uses a very specific internal mathematical model for tracking shapes, bounding boxes, and geometry. Because of this, importing completely arbitrary, complex SVG files generated by other tools is not fully supported. Elements like nested groups, advanced SVG filters (`<feDropShadow>`), masks, or complex relative coordinate paths may not be imported successfully or edit correctly within the HopeDraw interface.
